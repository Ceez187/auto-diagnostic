Minimum Live Data PID List For AutoMedic

Air/Fuel Core

STFT B1, LTFT B1, STFT B2, LTFT B2, MAF g/s, MAP kPa or MAP volts, calculated load, commanded EQ ratio or lambda if available.

O2 / A/F

Upstream O2/A/F sensors, downstream O2 sensors, O2 heater codes, cross-count/switching behavior, catalyst monitor data when available.

Electrical/No-Start

Battery/control module voltage, RPM while cranking, ECT, TPS/APP, fuel rail pressure if supported, security status, module communication/U-codes.
